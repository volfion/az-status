# Contributing to AZ Status

Thank you for helping improve AZ Status.

## Before opening an issue

- Search existing issues first.
- Include your macOS version and AZ Status version.
- Describe whether `http://azrouter.local` works in a browser.
- Never include passwords, tokens or private network details.

## Development

1. Fork and clone the repository.
2. Open `AZRouterMenu.xcodeproj` in Xcode.
3. Create a focused branch.
4. Keep changes small and related to one purpose.
5. Verify the app remains menu-bar-only and that settings update immediately.
6. Test both light and dark macOS appearance when changing UI.

## Pull requests

A pull request should include:

- a concise explanation of the change,
- testing steps,
- screenshots for visible UI changes,
- updated translations when user-facing text changes,
- an updated `CHANGELOG.md` entry when appropriate.
