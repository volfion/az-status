# Contributing

Contributions are welcome.

1. Fork the repository and create a focused branch.
2. Keep the app lightweight and menu-bar-first.
3. Do not commit credentials, API dumps containing personal data, signing certificates or provisioning profiles.
4. Build the Release configuration before opening a pull request.
5. Describe user-visible changes in `CHANGELOG.md`.

By contributing, you agree that your contribution is licensed under the MIT License.
