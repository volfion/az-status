# Privacy Policy

_Last updated: 16 July 2026_

AZ Status is designed for local, real-time monitoring.

- The app connects to the A-Z Router on the user's local network.
- The local password is stored in macOS Keychain.
- The app does not include analytics, advertising, telemetry or user tracking.
- The app does not upload photovoltaic-system measurements to the author.
- Choosing **Open Cloud** opens the A-Z Router cloud website in the default browser; that website is governed by its own privacy terms.
- macOS may deliver local connection-state notifications when the user enables them.

Questions can be submitted through the GitHub repository: `volfion/az-status`.
